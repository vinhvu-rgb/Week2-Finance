## Project Overview

This project simulates **multi-stock portfolio trading** using Python.  
It calculates **daily returns**, tracks **portfolio value**, and generates **visual plots** for each stock.  

This is an early-stage, beginner-friendly project but structured professionally to **show commitment, reproducibility, and real-world finance coding skills**.

---

## Features

- Load historical stock data from CSV files (`Date, Open, High, Low, Close, Volume`)  
- Calculate **daily returns** for each stock  
- Simulate simple **buy/sell trading logic** with **stop-loss**  
- Track portfolio value over time  
- Generate **visual plots** for each stock portfolio  
- Configurable: initial cash, stop-loss threshold, and stock list
